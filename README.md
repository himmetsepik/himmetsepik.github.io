# Himmet Sepik Links

A personal Linktree-style profile links page built with HTML, CSS and JavaScript.

## Features

- Modern dark design
- Mobile responsive layout
- Custom avatar
- Local logo assets
- Smooth button animation
- GitHub Pages support

## Links

- GitHub
- Codester
- Gumroad
- Instagram
- TikTok

## Technologies Used

- HTML
- CSS
- JavaScript
